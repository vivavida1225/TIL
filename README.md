## 📅 TIL 작성 원칙
1. **나의 언어로 재정리하기** - 단순히 공식 문서나 강의 코드를 받아적지 않습니다.
2. **이유를 기록하기** - 에러를 만났다면 '어떻게 해결했는지'보다 '왜 발생했고 어떻게 예방할지'를 적습니다.
3. **지속 가능하게 하기** - 너무 거창하게 쓰려다 지치지 않기 위해, 핵심 위주로 담백하게 기록합니다.
